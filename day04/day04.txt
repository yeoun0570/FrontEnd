1. rem, em 
    a. 글꼴의 크기 지정 단위 (상대 단위)
    b. 페이지의 크기(해상도)에 따라 글꼴의 크기가 달라진다.
    c. html 요소의 기본값이 16px = 1rem, 만약 기본값이 10px이면 10px = 1rem

2. px (절대단위)
    a. 페이지 크기가 줄든, 키우든 글꼴이 같은 크기를 갖는다.

3. overflow 속성
    a. 내용이 요소의 크기를 벗어났을때 어떻게 처리할지를 정하는 속성
    b. CSS에서는 기본값 visible(보여지다) 속성을 가진다.
    c. visible 속성을 hidden으로 바꿀 수 있다.
    d. hidden으로 바꾸면, 넘치는 컨텐츠가 보이지 않게된다.

4. overflow 속성 종류 (구글링 CSS overflow)
    a. visible
    b. hidden
    c. scroll
    d. auto

5. 배경이미지를 결정할 때 url
    a. background-img : url("") ==> "" 안에 이미지 주소를 지정
    b. border-radius : 요소의 모서리를 둥글게 만든다.
    c. max-width : 최대 너비 설정
    d. text-align : 텍스트 정렬 (left, right, center, justify(양쪽정렬))

6. CSS 클래스명 작성
    a. BEM : 부모의 클래스 이름을 이용하여 이름을 짓겠다.
    b. 구글링 (CSS BEM)
